# 🏛️ Histodle

**Histodle** es un juego de lógica y palabras inspirado en el famoso Wordle, pero enfocado exclusivamente en el mundo de la **Historia**. ¿Eres capaz de adivinar la palabra oculta antes de quedarte sin intentos?

---

## 🕹️ Cómo jugar

Jugar a Histodle es muy sencillo, pero requiere conocimientos históricos:

1.  **Adivina la palabra:** Tienes que escribir una palabra de **5 letras**. Todas están relacionadas con personajes, civilizaciones, lugares o conceptos históricos (ejemplo: *MAYAS*, *TROYA*, *LENIN*).
2.  **Introduce tu intento:** Escribe la palabra en el cuadro de texto y pulsa el botón **"Púlsame"**.
3.  **Analiza las pistas:** Después de cada intento, el color de las celdas cambiará para mostrarte qué tan cerca estás:
    * 🟩 **Verde:** La letra es correcta y está en la posición exacta.
    * 🟨 **Amarillo:** La letra está en la palabra, pero en una posición diferente.
    * 🟥 **Rojo:** La letra no está en la palabra.
4.  **Aprende mientras juegas:** ¡Lo mejor de Histodle! Cuando ganes o pierdas, aparecerá una **Explicación Histórica** detallada sobre la palabra del día para que siempre aprendas algo nuevo.

## 🚀 Instalación y uso local

Si quieres ejecutar este proyecto en tu propia computadora:

1.  Clona este repositorio:
    ```bash
    git clone [https://github.com/tu-usuario/histodle.git](https://github.com/tu-usuario/histodle.git)
    ```
2.  Entra en la carpeta:
    ```bash
    cd histodle
    ```
3.  Abre el archivo `proyecto.html` en tu navegador favorito. ¡Y listo para jugar!

## 🛠️ Tecnologías utilizadas

Este proyecto ha sido desarrollado "vainilla", es decir, sin frameworks pesados, para demostrar pura lógica de programación:
- **HTML5**: Estructura semántica.
- **CSS3**: Diseño responsivo y efectos visuales de colores.
- **JavaScript (ES6)**: Gestión del diccionario, lógica de validación de intentos y uso de `localStorage` para guardar tus victorias.
